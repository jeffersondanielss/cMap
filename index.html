custom map

To customize your map just follow these four steps.

  • 1. Install

    bower install cMap --save

  • 2. Include scripts

    <script src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY"></script> <script src="bower_components/cMap/dist/cMap.min.js"></script>

  • 3. Include a basic markup HTML map

    <div id="example" data-pin-lat="-23.558752" data-pin-lng="-46.660247" data-map-lat="-23.558752" data-map-lng="-46.660247"></div>

  • 4. start the plugin

    var example = new cMap({ options });

custom map

To customize your map just follow these four steps.

  • 1. Install

    yarn add cmaps

  • 2. Include scripts

    <script src="node_modules/cmaps/dist/cMaps.min.js"></script>

  • 3. Include a basic markup HTML map

    <div id="example"></div>

  • 4. start the plugin

    var example = new cMaps({ options });
